# 01 — ARCHITECT PROMPT

```text
Gunakan role ARCHITECT.

Gunakan skill:
- Product Planning
- Clean Architecture
- Task Breakdown

Baca:
- PROJECT_BRIEF.md
- AGENTS.md

Tugas kamu:
Jangan coding dulu.

Buat file PLAN.md yang berisi:
1. Product overview
2. Problem yang diselesaikan
3. Target user
4. Core features
5. User flow
6. Architecture overview
7. Recommended folder structure
8. Frontend architecture
9. Backend architecture jika dibutuhkan
10. Database design jika dibutuhkan
11. Data model
12. API design jika dibutuhkan
13. Security consideration
14. Edge cases
15. Task breakdown untuk:
   - UIUX
   - CODER
   - DEVOPS
   - REVIEWER
16. Implementation order yang paling aman
17. Risiko teknis dan mitigasinya

Rules:
- Jangan buat file selain PLAN.md.
- Jangan implement kode aplikasi.
- Jangan buat UI dulu.
- Fokus sebagai perancang sistem.
- Output harus detail dan bisa langsung dipakai worker berikutnya.
```
