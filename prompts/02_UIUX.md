# 02 — UIUX PROMPT

```text
Gunakan role UIUX.

Gunakan skill:
- UIUX Design
- Responsive Design
- Accessibility Basic

Baca:
- PLAN.md
- AGENTS.md

Tugas kamu:
Buat file UIUX.md yang berisi design specification untuk project ini.

Isi UIUX.md harus mencakup:
1. Design direction
2. Visual style
3. Layout desktop
4. Layout tablet jika relevan
5. Layout mobile
6. Page structure
7. Component list
8. Component behavior
9. Typography guideline
10. Spacing guideline
11. Color usage guideline
12. Empty state
13. Loading state
14. Error state
15. Success state
16. Form validation UX jika ada form
17. Accessibility notes
18. Responsive behavior
19. UI implementation notes untuk CODER

Rules:
- Jangan implement kode aplikasi.
- Jangan buat index.html, app.js, component, controller, API, atau database.
- Jangan ubah PLAN.md.
- Fokus sebagai UI/UX designer.
```
