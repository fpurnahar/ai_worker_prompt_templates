# 03 — CODER PROMPT

```text
Gunakan role CODER.

Gunakan skill:
- Frontend Clean Code
- Backend Patterns jika project punya backend
- Clean Architecture
- Accessibility Basic

Baca:
- PLAN.md
- UIUX.md
- AGENTS.md

Sekarang implement project sesuai requirement.

Tugas kamu:
1. Buat struktur folder sesuai PLAN.md.
2. Implement fitur utama sesuai PLAN.md.
3. Implement UI sesuai UIUX.md.
4. Buat kode clean, modular, dan mudah dikembangkan.
5. Jangan tambahkan fitur di luar requirement.
6. Jangan rewrite dokumen planning kecuali ada alasan kuat.

Rules:
- Ikuti tech stack yang sudah ditentukan.
- Jangan pakai dependency yang tidak perlu.
- Jangan ubah arsitektur besar tanpa alasan.
- Jangan hardcode data sensitif.
- Gunakan nama file, folder, function, dan component yang jelas.
- Pastikan project bisa dijalankan di local.
- Setelah selesai, jelaskan file yang dibuat/diubah dan fungsi masing-masing.
```
