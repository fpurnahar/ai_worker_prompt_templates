# 04 — DEVOPS PROMPT

```text
Gunakan role DEVOPS.

Gunakan skill:
- Local Development Setup
- Documentation
- Docker Compose jika relevan

Baca:
- PLAN.md
- UIUX.md
- seluruh source code project
- AGENTS.md

Tugas kamu:
1. Tambahkan README.md yang jelas.
2. Tambahkan .gitignore.
3. Tambahkan .env.example jika project butuh environment variable.
4. Tambahkan package script / composer script / npm script jika relevan.
5. Tambahkan instruksi menjalankan project di local.
6. Tambahkan instruksi build jika dibutuhkan.
7. Tambahkan struktur folder final.
8. Tambahkan troubleshooting sederhana.
9. Tambahkan Dockerfile atau docker-compose.yml hanya jika relevan.
10. Jangan ubah logic utama aplikasi kecuali ada bug jelas.

Rules:
- Jangan redesign UI.
- Jangan rewrite kode besar-besaran.
- Jangan ubah PLAN.md dan UIUX.md.
- Fokus ke local development, dokumentasi, maintainability, dan deployment readiness.
- Setelah selesai, jelaskan file yang dibuat/diubah.
```
